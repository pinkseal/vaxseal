# VaxSeal 🦭💉

Прививки, чекапы и донорство для эмигрантов в Сербии.
Узнай за 5 минут, какие прививки тебе нужны и где их сделать.

## Локальный запуск (если есть Node.js)

```bash
npm install
npm run dev
```

Открой http://localhost:5173

## Деплой на Vercel

1. Залей этот проект в GitHub-репозиторий
2. На vercel.com → Add New Project → выбери репозиторий
3. Vercel сам определит Vite — жми Deploy
4. Готово: получишь ссылку вида vaxseal.vercel.app

## Структура

- `src/App.jsx` — всё приложение (один файл)
- `src/main.jsx` — точка входа React
- `index.html` — HTML-шаблон с мета-тегами

## Данные

Хранятся локально в браузере пользователя (localStorage).
Бэкенда нет — приватность по умолчанию.

## Дисклеймер

Приложение даёт справочную информацию, не медицинские назначения.
Контент требует вычитки врачом перед публичным запуском.
